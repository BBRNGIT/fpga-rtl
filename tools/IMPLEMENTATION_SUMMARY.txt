ARCHITECTURE
============

Class Structure:
  - DeviceSpecMerger: Main class, handles all merging logic
  - SpecValue: Dataclass for spec with source/confidence
  - Conflict: Dataclass for conflict logging
  - Enums: Confidence, SourceType for type safety

Key Methods:
  - load_*_specs(): Load from each source
  - normalize_device_name(): Canonicalize device names
  - normalize_part_name(): Map vendor types to canonical
  - extract_numeric(): Parse numbers from strings
  - compare_numeric(): Cross-check with tolerance
  - merge_parts_list(): Reconcile parts across sources
  - infer_routing_capacity(): Estimate routing
  - validate_consistency(): Sanity checks
  - merge_device(): Merge single device
  - merge_all_devices(): Full merge
  - write_canonical_model(): Output JSON

Input Format (3 sources):
  device_specs.json: Datasheet (primary)
    {
      "source": "datasheet",
      "devices": {
        "xcvu9p-flga2104-2L": {
          "parts": [{"name": "CLB", "type": "CLB", "count": 182400}]
        }
      }
    }
  
  f4pga_devices.json: Community database
  trellis_ecp5.json: ECP5 bitstream reference

Output Format (canonical model):
  canonical_device_model.json
    {
      "schema_version": "1.0",
      "generation_timestamp": "...",
      "source_summary": {...},
      "devices": {
        "<device>": {
          "device": {name, sources, source_agreement},
          "parts": [{name, type, count{value,source,confidence}}],
          "routing": {local{...}, regional{...}},
          "validation": {consistency_score, conflicts, confidence},
          "audit": {conflicts_log[], warnings[]}
        }
      },
      "global_validation": {total_devices, high_confidence_devices, ...}
    }

USAGE EXAMPLES
==============

Sample Merge:
  $ python3 generate_sample_device_specs.py --output-dir test_data
  $ python3 merge_device_sources.py \
      --datasheet test_data/device_specs.json \
      --f4pga test_data/f4pga_devices.json \
      --trellis test_data/trellis_ecp5.json \
      --output test_data/canonical.json \
      -v

Python API:
  from merge_device_sources import DeviceSpecMerger
  merger = DeviceSpecMerger()
  merger.load_datasheet_specs("device_specs.json")
  merger.load_f4pga_specs("f4pga_devices.json")
  result = merger.merge_all_devices()
  merger.write_canonical_model("canonical.json")
