# Coherency and Interconnect (CCI-400)
Central coherent interconnect: ACE ports for APU/GPU/PL, DDR masters, SMMU DVM
**Type:** `interconnect`
- **data_width:** 128

## Connections
- **APU**: ACE (N/A bits, slave)
- **GPU**: ACE-Lite (N/A bits, slave)
- **DDR_CONTROLLER**: DDR_Master (N/A bits, master)
