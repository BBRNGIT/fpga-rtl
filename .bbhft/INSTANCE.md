# .bbhft — assembled FPGA instance (clone of THE one blank)

Identical instance of the immutable blank (per the factory law — a blank assigns
nothing at casting; this clone is named here, populated at assembly).

- fabric: VU9P blank (8,127,600 flip-flops, 1,182,000 LUTs, 120 GTY conductor lanes) — cloned byte-identical.
- name: .bbhft  (direct reference for the BIOS, placement, and adapters)
- status: Phase B (cloned, empty). Phase C assembles modules + buses onto it; Phase D installs the BIOS + adapters.
